# Bio-Breach / 바이오-브리치

> **2026 코리아 인디게임 데브캠프 출품작**
>
> 로우폴리 3D 생체 디펜스 × 복셀 기반 잠식 시뮬레이션

---

## 개요

**바이오-브리치(Bio-Breach)** 는 인체 내부를 전장으로 삼는 3인칭 협동 디펜스 게임입니다.
플레이어는 외계 바이러스 군집이 되어 숙주의 생체 조직을 실시간으로 해체하고, 그 자리에 변이 방어 유닛을 구축해 면역 세포의 역습을 막아냅니다.

| 항목 | 내용 |
|------|------|
| 장르 | 3인칭 협동 타워 디펜스 + 복셀 잠식 시뮬레이션 |
| 플랫폼 | PC |
| 플레이어 | 1~4인 Co-op (Host-Client) |
| 엔진 | Unity (Marching Cubes 기반 복셀 지형) |
| 네트워크 | Unity Netcode, Server Authority 모델 |

---

## 핵심 컨셉: 침략자이자 방어자의 역설

플레이어는 서사적으로는 숙주를 파괴하는 **침략자**이지만,
게임 매커니즘상으로는 자신의 **핵(Nucleus)** 을 지켜야 하는 **방어자**입니다.

- 파괴의 카타르시스와 수성의 긴장감을 동시에 제공
- 자원 채굴자 / 경로 설계자 / 전투 지휘관으로 나뉜 역할 분담 Co-op

## 게임플레이

### 복셀 잠식 시스템

32×32×32 Chunk 단위의 생체 조직을 **분해 도구(Dissolve Tool)** 로 실시간 파괴합니다.
파괴된 조직은 4종 자원으로 환원됩니다.

| 생체 조직 | VoxelType | 내구도 | 획득 자원 | 용도 |
|-----------|-----------|--------|-----------|------|
| 근육/결합 조직 | `Protein` | 1.0 | 단백질 | 변이 타워 건설 |
| 혈관/혈액 조직 | `Iron` | 1.5 | 철분 | 구조 보강 |
| 골격 조직 | `Calcium` | 3.0 | 칼슘 | 기본 능력·골격 강화 |
| 희귀 핵산 물질 | `GeneticEssence` | 10.0 | 유전자 정수 | 특수 기술 |

### 생체 조직 분포 레이어

```
표층 (y > -10)       : 단백질(기본) + 철분 혈관 덩어리 — 근육·혈관 지대
중층 (-30 < y ≤ -10) : 단백질(기본) + 칼슘 덩어리      — 연골 지대
심층 (y ≤ -30)       : 단백질(기본) + 칼슘 + 유전자 정수 극희귀 광맥 — 골수 지대
```

### 신경 에너지 & 괴사(Necrosis) 리스크

모든 변이 타워는 숙주 신경망의 에너지를 소모합니다.
**신경절 흡수기**로 공급망을 구축해야 하며, 공급망이 끊기면 후방 타워가 **괴사(비활성화)** 됩니다.

### 변이 방어 유닛 상성

| 유닛 | 특기 |
|------|------|
| 가시 세포 | 빠른 연사 → 백혈구 대응 |
| 포자 포탄 | 광역 부식 → 대식세포 군단 소탕 |
| 점액 트랩 | 이동 속도 감소 → Kill Zone 체류 시간 증가 |
| 마비 촉수 | 단일 타겟 무력화 → 엘리트 유닛 특화 |

---

## 적 유닛 & 3단계 페이즈

### 면역 세포 목록

| 유닛명 | 역할 | 특징 |
|--------|------|------|
| 백혈구 | 기본 보병 | 빠른 기동력, 근접 자폭 공격 |
| 대식세포 | 탱커 | 높은 체력, 타워 직접 삼킴 |
| 림프구 | 원거리 | 항체 미사일로 타워 정밀 타격 |
| 항생제 로봇 | 엘리트 | 약물 복용 시 투입, 압도적 화력 |

### 게임 루프 페이즈

```
1. 잠복기 (Incubation)    — 자원 수집 · 초기 인프라 구축 (빌더)
2. 발열기 (Fever)         — 상점 · 유전자 테크 트리 업그레이드 (정비)
3. 면역 폭풍 (Cytokine Storm) — 대규모 공세 + 지형 붕괴 동시 발생 (생존)
```

## 레벨 디자인 — 생체 지형 특수 구역

| 구역 | 특징 |
|------|------|
| 거대 동맥 (파괴 불가) | 적 고속도로 — 예측 가능한 주 이동 경로 |
| 모세혈관 (파괴 가능) | 파괴 시 혈소판·백혈구 급파 — 단기 자원 vs 장기 생존 딜레마 |
| 석회화된 뼈 | 절대 방어 구역 — 후반 특수 업그레이드로만 개척 가능 |
| 신경 다발 | 채굴 시 숙주 비명 + 화면 흔들림 + 전체 면역 유닛 광폭화 |

**조직 재생 시스템 (Anti-Cheese):** 숙주가 스스로 파괴된 지형을 메워 경로 왜곡 꼼수를 방지합니다.

## 기술 스택

| 기술 | 용도 |
|------|------|
| Unity + Marching Cubes | 유동적 생체 복셀 지형 생성 |
| Chunk 부분 재생성 | 수정된 Chunk만 Mesh 재생성 → CPU·네트워크 최소화 |
| Unity Netcode (Server Authority) | 4인 멀티플레이 지형 동기화 공정성 확보 |

---

## MVP 범위

- 유기물 테마 복셀 맵 1종
- 핵심 자원 4종 (단백질 · 철분 · 칼슘 · 유전자 정수)
- 변이 타워 3종 / 면역 세포 3종 밸런스 환경
- 3인칭 시점 2인 협동 플레이 + 핵심 동기화 구현
